# 📊 HR Data Analytics: Employee Attrition & Performance Analysis

## 📌 Project Overview
This project focuses on analyzing Human Resources (HR) data to understand the key factors driving employee attrition (leaving the company) and to evaluate workforce performance. Using a dataset of employee demographics, job roles, and satisfaction levels, I built an end-to-end data pipeline to deliver actionable insights for HR managers to improve retention.

## 🛠️ Tech Stack & Tools Used
* **Data Cleaning & Preprocessing:** Advanced Excel & Python (Pandas)
* **Database Management & Querying:** SQL (MySQL / PostgreSQL)
* **Data Visualization & Dashboarding:** Power BI & Tableau
  
## 📈 Data Analysis Workflow

### 1. Data Cleaning (Excel & Python)
* Handled missing values in employee tenure and department columns.
* Fixed inconsistent formatting in "Gender" and "Marital Status" fields.
* Created a new feature `Age_Group` (e.g., 20-30, 31-40) using Python for deeper demographic analysis.

### 2. Database & SQL Queries
I loaded the cleaned dataset into an SQL database and wrote optimization queries to identify trends:
* **Department-wise Attrition Rate:** Calculated which business units experience the highest turnover.
* **Salary vs Attrition:** Analyzed if employees with below-average monthly incomes leave faster.
* **Experience & Promotion:** Checked the average years since an employee's last promotion relative to their performance rating.

### 3. Interactive Dashboards (Power BI & Tableau)
Developed dynamic, interactive dashboards to track Key Performance Indicators (KPIs):
* **Key Metrics Tracked:** Total Employees, Active Employees, Attrition Rate (%), Average Age, and Average Monthly Income.
* **Visuals Included:**
  * Attrition by Department & Job Role (Bar Charts)
  * Job Satisfaction vs. Attrition (Matrix / Heatmaps)
  * Monthly Income vs. Attrition by Gender (Scatter Plot / Line Charts)
